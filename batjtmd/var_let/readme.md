## this

- this 是什么？
  函数**执行**的时候会立即生成的对象
  ，叫 this
  由调用方式决定
  指向最后的调用者
  普通函数方式运行，this 指向 window
  严格模式下，this 指向实例对象
  new 方式运行 ，this 指向实例对象
  对象的方法 this 对象
  事件处理函数来运行，指向事件对象
  箭头函数，本身没有 this 其实是作用域链中的 this
