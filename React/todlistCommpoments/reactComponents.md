## 组件化思想

- 现代前端开发框件的核心思想
- 低级 dom 树编程，--> 组件树编程
- 开发的最小单元
  html 只是沙子
  组件才是任务单元
- 组件组合一堆 html ,css,js 实现一个组合功能
  方便复用
- 组件组合在一起，完成页面开发
  页面由组件构成，现代前端就是用组件搭乐高积木
- 会写组件，会拆分组件，写数据状态，写前端页面
